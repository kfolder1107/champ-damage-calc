## 📋 プロジェクト概要

既存の Pokemon SV ダメージ計算ツール（`pokemon-sv-damage-calc`）をベースに、**Pokemon Champions（ポケモンチャンピオンズ）仕様のシングルバトル専用ダメージ計算ツール**として完全再設計する。

- **リポジトリ**: https://github.com/AdvanceHacker9361/pokemon-sv-damage-calc
- **参考ツール**: https://poke.terurun.jp/damage-calc （ダブルバトルメインだが計算精度・UX参考）
- **対象**: シングルバトル専用（ダブルバトル関連のUI・計算ロジックは除外）

## 🔧 Phase 2: Pokemon Champions 仕様差分の実装

### 2-1. ステータス計算式の変更

Pokemon Champions では従来のステータス計算式が以下のように変更されています：

```
【HP】
能力値 = (種族値×2 + 個体値31 + 能力ポイント×2) × 50 ÷ 100 + 60
※計算途中で小数点以下が生じる度に切り捨て

【HP以外】
能力値 = {(種族値×2 + 個体値31 + 能力ポイント×2) × 50 ÷ 100 + 5} × 性格補正
※性格補正: 上昇1.1、下降0.9、無補正1.0
※計算途中で小数点以下が生じる度に切り捨て
```

**実装指示：**
```
ステータス計算モジュールを以下のChampions仕様で再実装してください：

1. 個体値(IV)は全ポケモン一律31固定（UI上は非表示にし、内部計算で31を使用）
2. 従来の努力値(EV: 0-252, 合計510)を「能力ポイント(SP)」に置き換え：
   - 各ステータス: 0〜32 ポイント
   - 合計上限: 66 ポイント
   - 1ポイント = 実数値に直接+1（性格補正なしの場合）
   - 内部的には「能力ポイント×2」を従来の努力値の位置に代入して計算
3. レベルは50固定（UI上は非表示）
4. 性格補正あり能力のSPは、特定ポイントで+2される挙動を再現
   - 上昇補正の能力: 一部のポイントで実数値が+2
   - 下降補正の能力: 一部のポイントで+1に2ポイント必要
5. バリデーション: SP合計が66を超えた場合に警告表示

テストケース：
- ガブリアス(いじっぱり, A32/S32/H2): A実数値=200, S実数値=154, HP=184
- メガゲンガー(おくびょう, C32/S32/D2): C実数値=222, S実数値=200（メガ後種族値で計算）
```

### 2-2. ダメージ計算式の実装

```
ダメージ計算式はSV(第9世代)と基本同一であることが確認されています。
以下の計算式を正確に実装してください：

ダメージ = (攻撃側レベル×2÷5+2) → 切り捨て
         × 物理技(特殊技)の威力
         × 攻撃側のこうげき(とくこう) ÷ 防御側のぼうぎょ(とくぼう) → 切り捨て
         ÷ 50 + 2 → 切り捨て
         × 各種補正（以下の順番で逐次適用、各段階で五捨五超入 or 切り捨て）

補正の適用順序（重要）：
1. 天候補正 (晴れ: 炎1.5/水0.5, 雨: 水1.5/炎0.5, 砂嵐: 岩特防1.5)
2. 急所補正 (1.5倍)
3. 乱数 (0.85〜1.00 の16段階、各段階で切り捨て)
4. タイプ一致補正 (STAB: 1.5倍、てきおうりょく: 2.0倍)
5. タイプ相性 (0.25, 0.5, 1, 2, 4)
6. やけど補正 (物理技0.5倍)
7. その他補正（壁、持ち物、特性等）

16段階の乱数ロール(85%〜100%)すべてを計算し、以下を出力：
- 最小ダメージ〜最大ダメージ
- HP割合（最小%〜最大%）
- 確定n発 / 乱数n発（乱数の場合は確率も表示）
```

### 2-3. メガシンカ対応

```
Pokemon Champions のメガシンカシステムを実装してください：

2. メガシンカ時の種族値変更（元の種族値合計+100）
3. メガシンカ時のタイプ変更があるポケモンの対応

```

## ⚠️ 実装上の注意事項

### Champions 固有の注意点
- 個体値は完全に廃止（UI上も表示しない）
- SP(能力ポイント)は直感的で分かりやすいUIを心がける
- レベルは50固定で非表示
- PP体系は8/12/16/20の4段階に統一済み
- メガシンカは1バトル1回制限だが、ダメ計上は自由に切り替え可能にする
- ダイマックス・テラスタル・Zワザは存在しない
- 新規メガシンカのデータは攻略サイトの最新情報から取得

### 計算精度の注意点
- 全ての計算途中で小数点以下切り捨て（五捨五超入の箇所は仕様確認の上で実装）
- 乱数は0.85〜1.00の16段階（85, 86, 87, ..., 99, 100 を100で割る）
- 複数の補正が重なる場合の適用順序を厳密に守る
- 端数処理の差異で1ずれることがあるため、他ツールとの比較検証を徹底
